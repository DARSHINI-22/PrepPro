# PrepPro – AI-Powered Placement Preparation Platform

PrepPro is an AI-powered placement preparation platform developed using Google AI Studio to help students prepare for campus placements and competitive examinations.

The platform provides a unified environment for Data Structures & Algorithms practice, aptitude training, GATE CSE preparation, company-specific placement preparation, AI-powered mock interviews, language learning, progress tracking, and alumni networking.

## Features

* DSA Practice and Coding Preparation
* Aptitude and Logical Reasoning Training
* GATE CSE Preparation
* Foreign Languages - German and Japanese
* AI-Based Interview Assistance
* Progress Tracking Dashboard
* Alumni Connect and Career Guidance
* Student Networking and Community Hub
* Personalized Learning Recommendations

## Development Platform

* Google AI Studio
* Gemini AI Integration
* Modern Web-Based Interface

PrepPro aims to bridge the gap between academic learning and industry requirements by providing an intelligent and personalized placement preparation ecosystem.

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1o-Lwbs9WewRlUHHQ1gYzB-kwOooMGsGy

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
